# Virginia City Network
> Reconstructing the social structure of Montana's gold rush capital using historical data and network analysis.

## Project Overview
Virigina City Network is an interactive data science project that explores the relationships between residents, occupations, businesses, and families in Virginia City, Montana during the gold rush era.
By combining historical cencus records, newspapers, city directories, and archival records, this project aims to reconstruct the community as a social network and uncover patterns that traditional historical research may overlook.

## Research Questions
- How can network analysis unccover hidden relationships within Virginia City's mining community?
- Which occupations served as the largest community hubs?
- How were families, businesses, and organizations connected?
- Can network analysis reveal influential individuals who are largely forgotten today?

## Data Sources
- 1880 U.S. Census
- Historical newspapers
- Montana Historical Society archives
- City directories
- Cemetary records (if available)
